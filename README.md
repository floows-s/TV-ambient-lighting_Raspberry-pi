## Develop (Windows)
To generate visual studio project files run the folowing commands:
- mkdir VS-Project
- cd VS-Project
- cmake -G "Visual Studio 17 2022" ..

Next you can open up the *.sln file.

## Build (Linux)
To build the project run the following commands in the Build folder:
- mkdir Build
- cd Build
- cmake -G "Unix Makefile" ..
- make
